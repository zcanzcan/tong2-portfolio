# Admin Page Development Plan

## Goal
Develop a comprehensive admin page for the developer portfolio to manage dynamic content without code edits.

## Features

### 1. Authentication
- **Route**: `/admin`
- **Credentials**:
  - ID: `admin`
  - Password: `qor011zcan!`
- **UI**:
  - Subtle login button next to "Coffee Chat" in Hero section.

### 2. Profile Management
- Edit user's name.
- Edit profile information (title, bio, status).
- Edit contact details (email, phone).
- Edit resume link and coffee chat link.

### 3. Experience Management
- **CRUD Operations**: Create, Read, Update, Delete experience entries.
- **Fields**: Role, Company, Period, Color.

### 4. Skills (Tech Stack) Management
- **CRUD Operations**: Create, Read, Update, Delete skills.
- **Fields**: Skill name, etc.
