# 포트폴리오 관리 매뉴얼

이제 데이터가 Supabase DB를 통해 관리되므로, 로컬 파일을 수정하거나 배포용 스크립트를 실행할 필요가 없습니다.

## 1. 내용 수정 방법
실제 배포된 사이트 주소 뒤에 `/admin`을 붙여 접속하세요.
(예: `https://tong2-portfolio.vercel.app/admin`)

1. 관리자 아이디와 비밀번호로 로그인합니다.
2. 원하는 섹션을 수정하고 [저장] 버튼을 누릅니다.
3. 저장 즉시 실제 사이트에 반영됩니다.

## 2. 주의사항
- **데이터 백업**: 중요한 데이터는 수정 전 따로 메모해 두는 것을 권장합니다.
- **이미지 주소**: 이미지는 외부 링크 또는 Supabase Storage URL을 입력해 주세요.

## 참고사항
- 관리자 아이디/비번은 Vercel 환경변수에서 설정한 값입니다.
- `1_EDIT_PORTFOLIO.bat` 실행 시 검은색 창(서버 창)이 하나 더 뜹니다. 수정이 끝나면 그냥 닫으시면 됩니다.
