# 포트폴리오 관리 매뉴얼

Vercel 서버 특성상 웹에서 직접 저장이 불가능하므로, 아래 두 파일을 이용해 관리해 주세요.

## 1. 내용 수정할 때: `1_EDIT_PORTFOLIO.bat`
이 파일을 더블 클릭하면:
1. 자동으로 로컬 서버가 켜집니다.
2. 5초 뒤 크롬(브라우저)에서 관리자 페이지가 열립니다.
3. 여기서 자유롭게 내용을 수정하고 [저장] 하세요. (여기선 저장이 잘 됩니다!)

## 2. 수정을 마치고 배포할 때: `2_PUBLISH_CHANGES.bat`
수정이 다 끝나면 이 파일을 더블 클릭하세요:
1. 변경된 내용을 확인하고 엔터를 누르면
2. 자동으로 Vercel 서버로 전송됩니다.
3. 약 2~3분 뒤 실제 사이트(tong2-portfolio.vercel.app)가 업데이트됩니다.

## 참고사항
- 관리자 아이디/비번은 Vercel 환경변수에서 설정한 값입니다.
- `1_EDIT_PORTFOLIO.bat` 실행 시 검은색 창(서버 창)이 하나 더 뜹니다. 수정이 끝나면 그냥 닫으시면 됩니다.
